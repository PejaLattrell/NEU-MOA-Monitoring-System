🎯 Overview
The NEU-MOA-Monitoring-System is a full-featured web application designed to streamline and monitor administrative operations at Northeast University. The system provides real-time tracking, data management, and reporting capabilities through an intuitive user interface.

✨ Features
Real-time Monitoring: Live tracking and monitoring of administrative activities
Data Management: Comprehensive database management with Firestore
User Interface: Responsive and interactive web interface
Secure Access: Firebase authentication and security rules
Cloud Hosted: Deployed on Firebase hosting for reliability and scalability
🛠️ Tech Stack
Frontend: JavaScript (85.4%), HTML (10.2%), CSS (4.4%)
Backend: Firebase (Firestore, Authentication)
Database: Cloud Firestore
Hosting: Firebase Hosting
Package Manager: npm
